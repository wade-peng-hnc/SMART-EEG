隱私權政策

最後更新：2026-01-20

本應用程式（以下稱「本服務」）用於腦波資料上傳、分析與 SEA Index 指標呈現，並可透過 SMART on FHIR 與醫療系統整合。本政策說明我們如何蒐集、 使用與保護您的資料。

蒐集的資料類型

  • 使用者登入資訊（僅用於 SEA 服務登入與授權）。
  • 腦波資料檔案（.gz）與其中的基本欄位資訊。
  • FHIR 病患識別資訊（例如 Patient ID）。
  • 分析結果（SEA Index 指標）。

資料用途

  • 完成檔案上傳與分析流程。
  • 顯示 SEA Index 與相關狀態資訊。
  • 依 SMART on FHIR 規範將 Observation 寫入醫療系統。

資料分享與揭露

本服務僅在必要範圍內將資料傳送至既有的分析服務與 FHIR 伺服器。除非取得明確授權或法律要求，本服務不會將資料提供給第三方。

資料保存與安全

本服務不會在瀏覽器中長期保存敏感資訊，並避免於前端輸出包含個資的除錯日誌。 所有傳輸均依既有服務端點的安全規範進行。

您的權利

若您對資料的存取、更新或刪除有需求，請透過提供本服務的醫療機構或系統管理員處理。

聯絡方式

若有隱私權相關問題，請聯繫本服務管理單位或系統管理員。
